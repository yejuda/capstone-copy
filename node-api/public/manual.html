• 당신만을 위한 단 하나의 여행코스

추천받은 여행지가 별로일땐?
Planit - tips!

1) 인물보다는 배경이 부각되게!

             

얼굴 혹은 인물이 부각된 사진보다는
배경을 알아볼 수 있는 사진이 추천하기 원활합니다!


2) 이곳은 어떤 분위기? 확실한 분위기가 실린 사진을 올려주세요!

             

여행지의 분위기가 잘 실린 사진이 추천 받기 원활합니다!
Ex) 서울의 놀이공원을 추천하고 싶을땐?
왼쪽의 사진같이 올려주세요!


3) 초점을 맞춰주세요!

             

초점이 맞춰지지 않거나 흔들리는 사진은
제대로 된 여행지를 추천 받기 힘듭니다.


4) 사진 촬영도 밝게! 업로드 사진도 밝게!

             

사진의 밝기가 어두우면 다른 장소로 인식할 수 있어요!
